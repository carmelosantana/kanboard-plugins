# CalendarPlugin

A drag-and-drop calendar view for Kanboard, enabling visual task management by due date across all projects or per project. This plugin provides an intuitive calendar interface that seamlessly integrates with Kanboard's task management system.

## Installation

1. Clone or download this plugin into your Kanboard plugins directory.
2. Access the Kanboard admin panel and enable the CalendarPlugin.
3. The calendar view will appear as a new option in your project navigation.

## Requirements

- Kanboard >= 1.2.47
- PHP >= 8.4
