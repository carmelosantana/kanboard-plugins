# Kensho Plugin Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build `Kensho` — a private, per-user Kanboard reflection companion that mines the user's own activity, records who they want to be, and helps them reflect on alignment, with an LLM (via AiConnector) as a collaborator that proposes and questions, never an authority.

**Architecture:** A read-only data-mining model (`KenshoDataModel`) aggregates the current user's own tasks/categories/tags/time/activity into per-dimension signals. A per-user JSON store in `user_metadata` (`KenshoProfileModel`) holds user-defined life areas, importance, desired identity, goals, and saved reflections — no DB migration. Deterministic alignment math (`KenshoScoreModel`, pure static) turns areas + dimensions into importance/actual shares, discrepancies, and an optional bounded 0–100 score. A gate (`KenshoAiGate`) + AI model (`KenshoAiModel`) wrap AiConnector's `ProviderRegistry::structured()` for three opt-in calls (propose areas / clarify goals / reflect), degrading to fully manual when AiConnector is absent. A single controller + three template surfaces render the dashboard; all assets are buildless external JS + CSS styled with ShadcnTheme `kbx-*` primitives.

**Tech Stack:** PHP >= 8.4, Kanboard v1.2.47 core, PicoDb, AiConnector `ProviderRegistry` (consumed, never modified), ShadcnTheme `kbx-*` CSS primitives (consumed by class name only), PHPUnit (Kanboard `tests/units/Base.php`), buildless vanilla JS.

- **Isolation:** Create and edit ONLY `Kensho/`. Do NOT read or modify `CalendarPlugin/`, `TimeBlock/`, `ShadcnTheme/`, `AiConnector/`, `SubtaskGenerator/`, or any other plugin dir. Consume AiConnector's public API by the spec citations only.
- **`git init` `Kensho/` locally** (branch `main`, no remote). Commit after each task. Do NOT create a GitHub repo, push, tag, or release.
- **Harness edits are additive + UNCOMMITTED** (Task 8): append one line to `testing/docker-compose.dev.yml`, add `Kensho` to BOTH lists in `testing/run-plugin-tests.sh`. Never rewrite those files or touch other plugins' lines. All edits host-side (never `docker exec … sed/tee`).
- **plugin.json:** `name` "Kensho", `version` "1.0.0", `kanboard_version` ">=1.2.47", `php_version` ">=8.4", `homepage` "https://github.com/carmelosantana/kanboard-kensho", author "Carmelo Santana", license "MIT". `recommends`: `AiConnector >= 1.0.0` AND `ShadcnTheme >= 1.1.0`. **No `requires`.**
- **Buildless / CSP:** external `Assets/js` only, NO inline `<script>` (CSP `default-src 'self'`). Escape ALL template output with `$this->helper->text->e()` / `$this->e()`. Reusable-CSRF or form-CSRF for fetch endpoints.
- **AiConnector consumption:** instantiate `new \Kanboard\Plugin\AiConnector\Model\ProviderRegistry($this->container)`; guard with `class_exists(\Kanboard\Plugin\AiConnector\Model\ProviderRegistry::class)`. `structured($messages, $schemaJson, $profileId): array` is BLOCKING and returns a decoded array; `$schemaJson = json_encode($schema)`. Never log/echo API keys or full provider errors — log `get_class($e).' code '.$e->getCode()` only.
- **Privacy:** personal + self-hosted; AI is opt-in per run; send titles + aggregates + the user's own identity statements ONLY (no task descriptions/comments in v1). Show the "what will be sent" summary + provider picker before firing.
- **Product principles (BINDING, from spec §Product-principles):** No surface ever outputs "your purpose is X." Areas/groupings/importance/weights are the user's, contestable. Every AI artifact is editable and requires user confirmation to persist. Voice is gentle/non-judgmental; quiet/absent activity is an ordinary lapse, never failure. Prompts instruct the model to challenge rather than flatter and to avoid authoritative purpose/worth claims. Any numeric score ships with a permanent "chosen proxy, not a measurement of you" caveat.
- **Tests:** NO network/HTTP in unit tests — stub/mock `ProviderRegistry` or exercise the gate-closed path. Each test file starts `require_once 'tests/units/Base.php';` and `use KanboardTests\units\Base;`, class `extends Base`. Run with `./testing/run-plugin-tests.sh Kensho` from repo root.
- **Non-goals (do NOT build):** team/cross-user analytics; authoritative "your purpose is X"; nagging/notifications/cron; new time-capture; the four-circle ikigai Venn quiz; data egress beyond the user's chosen provider on an explicit action; descriptions/comments to the LLM; DB migration.

```php
use Kanboard\Core\Plugin\Base;

    {
```

```php
<?php

namespace Kanboard\Plugin\Kensho\Model;

use Kanboard\Plugin\AiConnector\Model\ProviderRegistry;

/**
 * KenshoAiGate — single source of truth for "is AI reflection available?".
 *
 * Gate = PHP >= 8.4 AND AiConnector present AND ProviderRegistry::isReady().
 * Consulted identically by Plugin/controller (to hide AI actions) and by every
 * AI route guard, so the UI and the guards never diverge. When closed, Kensho
 * stays fully functional in manual mode.
 */
class KenshoAiGate
{
    /**
     * @param \Pimple\Container $container
     * @param int|null  $phpVersionId     PHP_VERSION_ID override (tests).
     * @param bool|null $connectorPresent AiConnector-present override (tests).
     */
    public static function isReady($container, ?int $phpVersionId = null, ?bool $connectorPresent = null): bool
    {
        $versionId = $phpVersionId ?? PHP_VERSION_ID;
        if ($versionId < 80400) {
            return false;
        }

        $present = $connectorPresent
            ?? class_exists(\Kanboard\Plugin\AiConnector\Model\ProviderRegistry::class);
        if (! $present) {
            return false;
        }

        return (new ProviderRegistry($container))->isReady();
    }
}
```

Create `Kensho/Model/KenshoAiModel.php`:

```php
<?php

namespace Kanboard\Plugin\Kensho\Model;

use Kanboard\Core\Base;

/**
 * KenshoAiModel — the three opt-in AiConnector calls. Each builds a messages
 * array + a JSON Schema string, calls ProviderRegistry::structured() (BLOCKING,
 * returns a decoded array), and normalizes the result to a guaranteed shape so
 * the surfaces never crash on a malformed model response.
 *
 * Guardrails (spec §Product-principles) are encoded in every system prompt:
 *  - gentle, non-authoritative, second person;
 *  - CHALLENGE rather than flatter (counter sycophancy);
 *  - treat low/absent activity as an ordinary lapse, never a failure;
 *  - never claim the user's "purpose" or worth; defer to the user's own meaning.
 *
 * The AI proposes; the user disposes — nothing here persists anything.
 */
class KenshoAiModel extends Base
{
    /**
     * Reflect on computed discrepancies + the user's identity statements.
     * Returns gentle observations + questions the user answers themselves.
     * By construction there is NO verdict/score/rating field.
     *
     * @param array $discrepancies [{area_id, importance_share, actual_share, discrepancy}, ...]
     * @param array $identities    map area_id => desired-self statement
     * @return array{observations: string[], questions: string[], gentle_note: string}
     */
    public function reflect(array $discrepancies, array $identities, ?string $profileId = null): array
    {
        $schema = json_encode([
            'type'       => 'object',
            'properties' => [
                'observations' => ['type' => 'array', 'items' => ['type' => 'string']],
                'questions'    => ['type' => 'array', 'items' => ['type' => 'string']],
                'gentle_note'  => ['type' => 'string'],
            ],
            'required'   => ['observations', 'questions', 'gentle_note'],
        ]);

        $payload = [
            'discrepancies' => $discrepancies,
            'identities'    => $identities,
        ];

        $messages = [
            ['role' => 'system', 'content' => self::systemReflect()],
            ['role' => 'user',   'content' => json_encode($payload, JSON_PRETTY_PRINT | JSON_UNESCAPED_UNICODE)],
        ];

        $raw = $this->safeStructured($messages, $schema, $profileId);

        return [
            'observations' => self::stringList($raw['observations'] ?? []),
            'questions'    => self::stringList($raw['questions'] ?? []),
            'gentle_note'  => is_string($raw['gentle_note'] ?? null) ? $raw['gentle_note'] : '',
        ];
    }

    // ── Shared helpers ────────────────────────────────────────────────────────

    /**
     * Call structured() defensively. Returns [] on any provider error — the
     * caller normalizes []. Never logs the message (may carry provider details).
     */
```

```php
<?php

namespace Kanboard\Plugin\Kensho\Model;

use Kanboard\Core\Base;

/**
 * KenshoProfileModel — per-user Kensho profile stored as JSON in user_metadata
 * (key `kensho_profile`). No DB migration. Decoding is defensive: corrupt or
 * absent JSON, or a partial object, always yields a complete default-filled
 * profile so the surfaces never see a missing key.
 */
class KenshoProfileModel extends Base
{
    const KEY     = 'kensho_profile';
    const VERSION = 1;

    public static function defaults(): array
    {
        return [
            'version'     => self::VERSION,
            'window_days' => 90,
            'settings'    => ['show_score' => true],
            'areas'       => [],
            'reflections' => [],
        ];
    }

    public function getProfile(int $userId): array
    {
        $raw = (string) $this->userMetadataModel->get($userId, self::KEY, '');
        $data = $raw !== '' ? json_decode($raw, true) : null;
        if (! is_array($data)) {
            return self::defaults();
        }
        return self::sanitize($data);
    }

    public function saveProfile(int $userId, array $profile): bool
    {
        $clean = self::sanitize($profile);
        return (bool) $this->userMetadataModel->save($userId, [
            self::KEY => json_encode($clean, JSON_UNESCAPED_UNICODE),
        ]);
    }

    public function upsertArea(int $userId, array $area): array
    {
        $profile = $this->getProfile($userId);
        $area = self::sanitizeArea($area, $profile['areas']);

        $found = false;
        foreach ($profile['areas'] as $i => $existing) {
            if ($existing['id'] === $area['id']) {
                $profile['areas'][$i] = $area;
                $found = true;
                break;
            }
        }
        if (! $found) {
            $profile['areas'][] = $area;
        }

        $this->saveProfile($userId, $profile);
        return $this->getProfile($userId);
    }

    public function removeArea(int $userId, string $areaId): array
    {
        $profile = $this->getProfile($userId);
        $profile['areas'] = array_values(array_filter(
            $profile['areas'],
            fn ($a) => $a['id'] !== $areaId
        ));
        $this->saveProfile($userId, $profile);
        return $this->getProfile($userId);
    }

    public function addReflection(int $userId, array $reflection): array
    {
        $profile = $this->getProfile($userId);
        $profile['reflections'][] = self::sanitizeReflection($reflection);
        $this->saveProfile($userId, $profile);
        return $this->getProfile($userId);
    }

    // ── Sanitizers (defensive; fill defaults, clamp ranges) ────────────────────

    private static function sanitize(array $data): array
    {
        $d = self::defaults();
        $out = [
            'version'     => self::VERSION,
            'window_days' => in_array((int) ($data['window_days'] ?? $d['window_days']), [30, 90, 180, 365], true)
                ? (int) $data['window_days'] : 90,
            'settings'    => [
                'show_score' => (bool) ($data['settings']['show_score'] ?? true),
            ],
            'areas'       => [],
            'reflections' => [],
        ];

        if (isset($data['areas']) && is_array($data['areas'])) {
            foreach ($data['areas'] as $a) {
                if (is_array($a)) {
                    $out['areas'][] = self::sanitizeArea($a, $out['areas']);
                }
            }
        }
        if (isset($data['reflections']) && is_array($data['reflections'])) {
            foreach ($data['reflections'] as $r) {
                if (is_array($r)) {
                    $out['reflections'][] = self::sanitizeReflection($r);
                }
            }
        }
        return $out;
    }

    private static function sanitizeArea(array $a, array $siblings): array
    {
        $label = trim((string) ($a['label'] ?? ''));
        $id = trim((string) ($a['id'] ?? ''));
        if ($id === '') {
            $id = self::uniqueSlug($label, $siblings);
        }
        $importance = (int) ($a['importance'] ?? 5);
        $importance = max(1, min(10, $importance));

        $goals = [];
        if (isset($a['goals']) && is_array($a['goals'])) {
            foreach ($a['goals'] as $g) {
                if (is_string($g) && trim($g) !== '') {
                    $goals[] = trim($g);
                }
            }
        }

        $srcRaw = is_array($a['sources'] ?? null) ? $a['sources'] : [];
        $sources = [
            'project_ids'  => self::intList($srcRaw['project_ids'] ?? []),
            'category_ids' => self::intList($srcRaw['category_ids'] ?? []),
            'tag_ids'      => self::intList($srcRaw['tag_ids'] ?? []),
        ];

        $weight = isset($a['weight']) && is_numeric($a['weight']) ? (float) $a['weight'] : 1.0;
        if ($weight < 0) {
            $weight = 1.0;
        }

        return [
            'id'         => $id,
            'label'      => $label,
            'importance' => $importance,
            'identity'   => trim((string) ($a['identity'] ?? '')),
            'goals'      => $goals,
            'weight'     => $weight,
            'sources'    => $sources,
        ];
    }

    private static function sanitizeReflection(array $r): array
    {
        return [
            'id'               => trim((string) ($r['id'] ?? ('r_' . substr(md5(json_encode($r)), 0, 10)))),
            'date'             => (int) ($r['date'] ?? 0),
            'window_days'      => (int) ($r['window_days'] ?? 90),
            'score'            => isset($r['score']) && is_numeric($r['score']) ? (int) $r['score'] : null,
            'discrepancies'    => is_array($r['discrepancies'] ?? null) ? $r['discrepancies'] : [],
            'narrative'        => (string) ($r['narrative'] ?? ''),
            'provider_profile' => (string) ($r['provider_profile'] ?? ''),
        ];
    }

    private static function intList($value): array
    {
        if (! is_array($value)) {
            return [];
        }
        $out = [];
        foreach ($value as $v) {
            if (is_numeric($v)) {
                $out[] = (int) $v;
            }
        }
        return array_values(array_unique($out));
    }

    private static function uniqueSlug(string $label, array $siblings): string
    {
        $base = preg_replace('/[^a-z0-9]+/', '_', strtolower($label));
        $base = trim($base, '_');
        if ($base === '') {
            $base = 'area';
        }
        $slug = 'area_' . $base;
        $existing = array_column($siblings, 'id');
        $candidate = $slug;
        $n = 2;
        while (in_array($candidate, $existing, true)) {
            $candidate = $slug . '_' . $n;
            $n++;
        }
        return $candidate;
    }
}
```

```php
<?php

namespace Kanboard\Plugin\Kensho\Model;

/**
 * KenshoScoreModel — deterministic, transparent alignment math (spec §5).
 *
 * Turns user-defined areas + aggregated activity dimensions into per-area
 * importance-share vs actual-effort-share, a signed discrepancy, and an OPTIONAL
 * bounded 0–100 composite score (= 100 × the overlap between the intended and
 * actual effort distributions). Pure functions, no I/O — the whole thing is a
 * chosen proxy the user can inspect and contest, never a measurement of worth.
 *
 * Assignment: each dimension is claimed by the FIRST area (in order) whose
 * sources contain its (type,id); unclaimed dimensions form an implicit
 * "Unassigned" bucket. This guarantees Σ actual_share = 100 with no double count.
 */
class KenshoScoreModel
{
    const UNASSIGNED_ID = '__unassigned__';

    public static function alignment(array $areas, array $dimensions): array
    {
        // 1. Effort per dimension (time, or task_count fallback when no time logged).
        $totalTime = 0.0;
        foreach ($dimensions as $d) {
            $totalTime += (float) ($d['time_spent'] ?? 0);
        }
        $useCount = $totalTime <= 0.0;

        // 2. Assign each dimension to the first claiming area.
        $areaEffort = [];
        foreach ($areas as $a) {
            $areaEffort[$a['id']] = 0.0;
        }
        $unassignedEffort = 0.0;
        $totalEffort = 0.0;

        foreach ($dimensions as $d) {
            $effort = $useCount ? (float) ($d['task_count'] ?? 0) : (float) ($d['time_spent'] ?? 0);
            $totalEffort += $effort;
            $ownerId = self::firstOwner($areas, (string) ($d['type'] ?? ''), (int) ($d['id'] ?? 0));
            if ($ownerId === null) {
                $unassignedEffort += $effort;
            } else {
                $areaEffort[$ownerId] += $effort;
            }
        }

        // 3. Importance denominator.
        $impDenom = 0.0;
        foreach ($areas as $a) {
            $impDenom += (float) $a['importance'] * (float) ($a['weight'] ?? 1.0);
        }

        // 4. Build rows.
        $rows = [];
        foreach ($areas as $a) {
            $effort = $areaEffort[$a['id']];
            $actualShare = $totalEffort > 0 ? 100.0 * $effort / $totalEffort : 0.0;
            $impShare = $impDenom > 0
                ? 100.0 * ((float) $a['importance'] * (float) ($a['weight'] ?? 1.0)) / $impDenom
                : 0.0;
            $rows[] = [
                'area_id'          => $a['id'],
                'label'            => $a['label'],
                'importance'       => (int) $a['importance'],
                'weight'           => (float) ($a['weight'] ?? 1.0),
                'effort'           => $effort,
                'actual_share'     => round($actualShare, 1),
                'importance_share' => round($impShare, 1),
                'discrepancy'      => round($impShare - $actualShare, 1),
            ];
        }
        if ($unassignedEffort > 0) {
            $actualShare = $totalEffort > 0 ? 100.0 * $unassignedEffort / $totalEffort : 0.0;
            $rows[] = [
                'area_id'          => self::UNASSIGNED_ID,
                'label'            => 'Unassigned',
                'importance'       => 0,
                'weight'           => 0.0,
                'effort'           => $unassignedEffort,
                'actual_share'     => round($actualShare, 1),
                'importance_share' => 0.0,
                'discrepancy'      => round(0.0 - $actualShare, 1),
            ];
        }

        // 5. Composite score (opt-in at the UI; computed here when meaningful).
        $score = null;
        if (! empty($areas) && $totalEffort > 0 && $impDenom > 0) {
            $sumAbs = 0.0;
            foreach ($rows as $row) {
                $sumAbs += abs($row['importance_share'] - $row['actual_share']);
            }
            $score = (int) round(100.0 - 0.5 * $sumAbs);
            $score = max(0, min(100, $score));
        }

        return [
            'uses_task_count_fallback' => $useCount && $totalEffort > 0,
            'total_effort'             => $totalEffort,
            'rows'                     => $rows,
            'score'                    => $score,
        ];
    }

    /** First area (in order) whose sources contain (type,id); null if none. */
    private static function firstOwner(array $areas, string $type, int $id): ?string
    {
        $key = match ($type) {
            'project'  => 'project_ids',
            'category' => 'category_ids',
            'tag'      => 'tag_ids',
            default    => null,
        };
        if ($key === null) {
            return null;
        }
        foreach ($areas as $a) {
            $ids = $a['sources'][$key] ?? [];
            if (is_array($ids) && in_array($id, array_map('intval', $ids), true)) {
                return $a['id'];
            }
        }
        return null;
    }
```

```php
<?php

namespace Kanboard\Plugin\Kensho\Model;

use Kanboard\Core\Base;
use Kanboard\Model\TaskModel;
use Kanboard\Model\ProjectActivityModel;

/**
 * KenshoDataModel — read-only aggregation of the CURRENT user's OWN activity
 * into per-dimension signals (project / category / tag). Window-filtered by
 * date_modification and access-filtered by getActiveProjectIds(). No writes.
 *
 * Only aggregated patterns are meaningful — a single task or a quiet week is a
 * weak signal, so callers must never infer identity from one row.
 */
class KenshoDataModel extends Base
{
    public function aggregate(int $userId, int $windowDays): array
    {
        $windowDays = max(1, $windowDays);
        $cutoff = time() - ($windowDays * 86400);

        $projectIds = $this->projectPermissionModel->getActiveProjectIds($userId);
        if (empty($projectIds)) {
            return ['window_days' => $windowDays, 'active_days' => 0, 'dimensions' => []];
        }

        // The user's own tasks in accessible projects within the window (open + closed).
        $tasks = $this->db->table(TaskModel::TABLE)
            ->columns('id', 'project_id', 'category_id', 'time_spent', 'time_estimated', 'is_active', 'date_modification', 'title')
            ->eq('owner_id', $userId)
            ->in('project_id', $projectIds)
            ->gte('date_modification', $cutoff)
            ->findAll();

        $taskIds = array_map(fn ($t) => (int) $t['id'], $tasks);
        $tagsByTask = empty($taskIds) ? [] : $this->taskTagModel->getTagsByTaskIds($taskIds);

        $projectLabels = $this->projectModel->getList(false);
        $categoryLabels = $this->categoryLabels($tasks);

        // Accumulate per dimension key "type:id".
        $acc = [];
        $tagMeta = [];
        foreach ($tasks as $t) {
            $day = (int) floor(((int) $t['date_modification']) / 86400);

            $this->bump($acc, 'project', (int) $t['project_id'], $projectLabels[(int) $t['project_id']] ?? ('#' . $t['project_id']), $t, $day);

            if ((int) $t['category_id'] > 0) {
                $this->bump($acc, 'category', (int) $t['category_id'], $categoryLabels[(int) $t['category_id']] ?? ('#' . $t['category_id']), $t, $day);
            }

            foreach ($tagsByTask[(int) $t['id']] ?? [] as $tag) {
                $tagMeta[(int) $tag['id']] = $tag['name'];
                $this->bump($acc, 'tag', (int) $tag['id'], $tag['name'], $t, $day);
            }
        }

        $dimensions = [];
        foreach ($acc as $entry) {
            $dimensions[] = [
                'type'            => $entry['type'],
                'id'              => $entry['id'],
                'label'           => $entry['label'],
                'task_count'      => $entry['task_count'],
                'time_spent'      => round($entry['time_spent'], 2),
                'time_estimated'  => round($entry['time_estimated'], 2),
                'completion_rate' => $entry['task_count'] > 0 ? round($entry['closed'] / $entry['task_count'], 3) : 0.0,
                'active_days'     => count($entry['days']),
                'sample_titles'   => array_slice($entry['titles'], 0, 5),
            ];
        }

        return [
            'window_days' => $windowDays,
            'active_days' => $this->activeDays($userId, $cutoff),
            'dimensions'  => $dimensions,
        ];
    }

    /** Distinct days with any activity by this user in the window. */
    private function activeDays(int $userId, int $cutoff): int
    {
        $rows = $this->db->table(ProjectActivityModel::TABLE)
            ->columns('date_creation')
            ->eq('creator_id', $userId)
            ->gte('date_creation', $cutoff)
            ->findAll();
        $days = [];
        foreach ($rows as $r) {
            $days[(int) floor(((int) $r['date_creation']) / 86400)] = true;
        }
        return count($days);
    }

    /** id => name for every category id referenced by the task set. */
    private function categoryLabels(array $tasks): array
    {
        $ids = [];
        foreach ($tasks as $t) {
            if ((int) $t['category_id'] > 0) {
                $ids[(int) $t['category_id']] = true;
            }
        }
        if (empty($ids)) {
            return [];
        }
        $rows = $this->db->table('categories')
            ->columns('id', 'name')
            ->in('id', array_keys($ids))
            ->findAll();
        $out = [];
        foreach ($rows as $r) {
            $out[(int) $r['id']] = $r['name'];
        }
        return $out;
    }

    private function bump(array &$acc, string $type, int $id, string $label, array $task, int $day): void
    {
        $key = $type . ':' . $id;
        if (! isset($acc[$key])) {
            $acc[$key] = [
                'type' => $type, 'id' => $id, 'label' => $label,
                'task_count' => 0, 'time_spent' => 0.0, 'time_estimated' => 0.0,
                'closed' => 0, 'days' => [], 'titles' => [],
            ];
        }
        $acc[$key]['task_count']++;
        $acc[$key]['time_spent'] += (float) $task['time_spent'];
        $acc[$key]['time_estimated'] += (float) $task['time_estimated'];
        if ((int) $task['is_active'] === 0) {
            $acc[$key]['closed']++;
        }
        $acc[$key]['days'][$day] = true;
        if (count($acc[$key]['titles']) < 5) {
            $acc[$key]['titles'][] = (string) $task['title'];
        }
    }
}
```

```php
     * Propose candidate life-area groupings from the aggregated dimensions.
     * Input carries dimension labels + signals + up to 5 sample TITLES only
     * (no descriptions/comments). Output is a scaffold the user edits/confirms.
     *
     * @return array{areas: array<int, array{label:string, rationale:string, dimension_refs: array<int, array{type:string,id:int}>, importance_hint: int|null}>}

    public function proposeAreas(array $aggregate, ?string $profileId = null): array
    {
        $schema = json_encode([
            'type'       => 'object',
            'properties' => [
                'areas' => [
                    'type'  => 'array',
                    'items' => [
                        'type'       => 'object',
                        'properties' => [
                            'label'          => ['type' => 'string'],
                            'rationale'      => ['type' => 'string'],
                            'dimension_refs' => [
                                'type'  => 'array',
                                'items' => [
                                    'type'       => 'object',
                                    'properties' => [
                                        'type' => ['type' => 'string', 'enum' => ['project', 'category', 'tag']],
                                        'id'   => ['type' => 'integer'],
                                    ],
                                    'required'   => ['type', 'id'],
                                ],
                            ],
                            'importance_hint' => ['type' => 'integer'],
                        ],
                        'required'   => ['label', 'rationale', 'dimension_refs'],
                    ],
                ],
            ],
            'required'   => ['areas'],
        ]);

        $messages = [
            ['role' => 'system', 'content' => self::systemPropose()],
            ['role' => 'user',   'content' => json_encode($aggregate, JSON_PRETTY_PRINT | JSON_UNESCAPED_UNICODE)],
        ];

        $raw = $this->safeStructured($messages, $schema, $profileId);
        $areas = [];
        foreach (($raw['areas'] ?? []) as $a) {
            if (! is_array($a)) {
                continue;
            }
            $label = trim((string) ($a['label'] ?? ''));
            if ($label === '') {
                continue;
            }
            $refs = [];
            foreach (($a['dimension_refs'] ?? []) as $r) {
                if (is_array($r) && isset($r['type'], $r['id'])) {
                    $refs[] = ['type' => (string) $r['type'], 'id' => (int) $r['id']];
                }
            }
            $areas[] = [
                'label'           => $label,
                'rationale'       => trim((string) ($a['rationale'] ?? '')),
                'dimension_refs'  => $refs,
                'importance_hint' => isset($a['importance_hint']) && is_numeric($a['importance_hint'])
                    ? max(1, min(10, (int) $a['importance_hint'])) : null,
            ];
        }

     * concrete goals + one reflection question. Edit-before-save scaffold.

            ['role' => 'system', 'content' => self::systemClarify()],
            ['role' => 'user',   'content' => json_encode(['area' => $areaLabel, 'note' => $note], JSON_UNESCAPED_UNICODE)],
        ];

        $raw = $this->safeStructured($messages, $schema, $profileId);
        return [
            'identity'            => is_string($raw['identity'] ?? null) ? trim($raw['identity']) : '',
            'goals'               => self::stringList($raw['goals'] ?? []),
```

Create `Kensho/Controller/KenshoController.php`:

```php
<?php

namespace Kanboard\Plugin\Kensho\Controller;

use Kanboard\Controller\BaseController;
use Kanboard\Core\Controller\AccessForbiddenException;
use Kanboard\Plugin\Kensho\Model\KenshoAiGate;
use Kanboard\Plugin\Kensho\Model\KenshoAiModel;
use Kanboard\Plugin\Kensho\Model\KenshoDataModel;
use Kanboard\Plugin\Kensho\Model\KenshoProfileModel;
use Kanboard\Plugin\Kensho\Model\KenshoScoreModel;

/**
 * KenshoController — personal reflection dashboard + mutations + AI actions.
 *
 * Every action operates on the current user's OWN data (userSession id); there
 * is no project-admin gate (login is enforced by the router). Every mutation
 * calls checkCSRFForm(); the three AI actions additionally require the gate to
 * be open and re-check it (degrade to manual when AiConnector is absent).
 */
class KenshoController extends BaseController
{
    // ── Dashboard ──────────────────────────────────────────────────────────────

    public function show(): void
    {
        $userId = $this->userSession->getId();
        $profile = $this->profileModel()->getProfile($userId);
        $aggregate = $this->dataModel()->aggregate($userId, $profile['window_days']);
        $alignment = KenshoScoreModel::alignment($profile['areas'], $aggregate['dimensions']);

        $aiReady = $this->isAiEnabled();
        $profiles = [];
        $defaultProfileId = '';
        if ($aiReady) {
            $registry = new \Kanboard\Plugin\AiConnector\Model\ProviderRegistry($this->container);
            $profiles = $registry->listProfiles();
            $defaultProfileId = $registry->getDefaultProfileId();
        }

        $this->response->html($this->helper->layout->app('Kensho:dashboard/show', [
            'title'              => t('Kensho'),
            'profile'            => $profile,
            'aggregate'          => $aggregate,
            'alignment'          => $alignment,
            'ai_ready'           => $aiReady,
            'profiles'           => $profiles,
            'default_profile_id' => $defaultProfileId,
        ]));
    }

    // ── Mutations (CSRF on every one) ──────────────────────────────────────────

    public function analyze(): void
    {
        $this->checkCSRFForm();
        $userId = $this->userSession->getId();
        $profile = $this->profileModel()->getProfile($userId);
        $profile['window_days'] = (int) $this->request->getIntegerParam('window_days', $profile['window_days']);
        $profile['settings']['show_score'] = $this->request->getIntegerParam('show_score', $profile['settings']['show_score'] ? 1 : 0) === 1;
        $this->profileModel()->saveProfile($userId, $profile);
        $this->response->redirect($this->helper->url->to('KenshoController', 'show', ['plugin' => 'Kensho']));
    }

    public function areas(): void
    {
        $this->checkCSRFForm();
        $userId = $this->userSession->getId();
        $areas = $this->decodeJsonParam('areas');

        $profile = $this->profileModel()->getProfile($userId);
        $profile['areas'] = is_array($areas) ? $areas : [];
        $this->profileModel()->saveProfile($userId, $profile); // saveProfile sanitizes each area
        $this->flash->success(t('Your areas were saved.'));
        $this->response->redirect($this->helper->url->to('KenshoController', 'show', ['plugin' => 'Kensho']));
    }

    public function goals(): void
    {
        $this->checkCSRFForm();
        $userId = $this->userSession->getId();
        $updates = $this->decodeJsonParam('goals'); // [{id, importance, identity, goals:[]}, ...]

        if (is_array($updates)) {
            foreach ($updates as $u) {
                if (! is_array($u) || empty($u['id'])) {
                    continue;
                }
                $profile = $this->profileModel()->getProfile($userId);
                foreach ($profile['areas'] as $existing) {
                    if ($existing['id'] === $u['id']) {
                        $merged = array_merge($existing, [
                            'importance' => (int) ($u['importance'] ?? $existing['importance']),
                            'identity'   => (string) ($u['identity'] ?? $existing['identity']),
                            'goals'      => is_array($u['goals'] ?? null) ? $u['goals'] : $existing['goals'],
                        ]);
                        $this->profileModel()->upsertArea($userId, $merged);
                        break;
                    }
                }
            }
        }
        $this->flash->success(t('Your goals were saved.'));
        $this->response->redirect($this->helper->url->to('KenshoController', 'show', ['plugin' => 'Kensho']));
    }

    public function reflection(): void
    {
        $this->checkCSRFForm();
        $userId = $this->userSession->getId();
        $this->profileModel()->addReflection($userId, [
            'date'             => time(),
            'window_days'      => (int) $this->request->getIntegerParam('window_days', 90),
            'score'            => $this->request->getStringParam('score', '') !== '' ? (int) $this->request->getIntegerParam('score', 0) : null,
            'discrepancies'    => $this->decodeJsonParam('discrepancies') ?: [],
            'narrative'        => (string) $this->request->getStringParam('narrative', ''),
            'provider_profile' => (string) $this->request->getStringParam('provider_profile', ''),
        ]);
        $this->flash->success(t('Reflection saved.'));
        $this->response->redirect($this->helper->url->to('KenshoController', 'show', ['plugin' => 'Kensho']));
    }

    // ── AI actions (CSRF + gate; JSON responses) ──────────────────────────────

    public function propose(): void
    {
        $this->requireAi();
        $this->checkCSRFForm();
        $userId = $this->userSession->getId();
        $profile = $this->profileModel()->getProfile($userId);
        $aggregate = $this->dataModel()->aggregate($userId, $profile['window_days']);
        $out = $this->aiModel()->proposeAreas($aggregate, $this->resolveProfileId());
        $this->response->json($out);
    }

    public function clarify(): void
    {
        $this->requireAi();
        $this->checkCSRFForm();
        $out = $this->aiModel()->clarifyGoals(
            (string) $this->request->getStringParam('area_label', ''),
            (string) $this->request->getStringParam('note', ''),
            $this->resolveProfileId()
        );
        $this->response->json($out);
    }

    public function reflect(): void
    {
        $this->requireAi();
        $this->checkCSRFForm();
        $userId = $this->userSession->getId();
        $profile = $this->profileModel()->getProfile($userId);
        $aggregate = $this->dataModel()->aggregate($userId, $profile['window_days']);
        $alignment = KenshoScoreModel::alignment($profile['areas'], $aggregate['dimensions']);

        $identities = [];
        foreach ($profile['areas'] as $a) {
            if ($a['identity'] !== '') {
                $identities[$a['id']] = $a['identity'];
            }
        }
        $out = $this->aiModel()->reflect($alignment['rows'], $identities, $this->resolveProfileId());
        $this->response->json($out);
    }

    // ── Assets (buildless; served with correct content-type) ───────────────────

    public function css(): void
    {
        $this->response->css(file_get_contents(__DIR__ . '/../Assets/css/kensho.css'));
    }

    public function js(): void
    {
        $this->response->js(file_get_contents(__DIR__ . '/../Assets/js/kensho.js'));
    }

    // ── Helpers ────────────────────────────────────────────────────────────────

    /** Reject with 403 unless the AI gate is open. Mirrors GeneratorController. */
    private function requireAi(): void
    {
        if (! $this->isAiEnabled()) {
            throw new AccessForbiddenException();
        }
    }

    protected function isAiEnabled(): bool
    {
        return KenshoAiGate::isReady($this->container);
    }

    /** Validate the posted provider profile id against known ids (else null). */
    private function resolveProfileId(): ?string
    {
        $profileId = trim((string) $this->request->getStringParam('provider_profile', ''));
        if ($profileId === '') {
            return null;
        }
        $registry = new \Kanboard\Plugin\AiConnector\Model\ProviderRegistry($this->container);
        $known = array_column($registry->listProfiles(), 'id');
        return in_array($profileId, $known, true) ? $profileId : null;
    }

    private function decodeJsonParam(string $name)
    {
        $raw = (string) $this->request->getStringParam($name, '');
        if ($raw === '') {
            return null;
        }
        $decoded = json_decode($raw, true);
        return is_array($decoded) ? $decoded : null;
    }

    protected function profileModel(): KenshoProfileModel
    {
        return new KenshoProfileModel($this->container);
    }

    protected function dataModel(): KenshoDataModel
    {
        return new KenshoDataModel($this->container);
    }

    protected function aiModel(): KenshoAiModel
    {
        return new KenshoAiModel($this->container);
    }
}
```

- [ ] **Step 10: Buildless external JS**

Create `Kensho/Assets/js/kensho.js`. Full behavior: tab switching; area editor (add/edit/remove area rows, collect into JSON, POST to `kensho/areas` via a form); goals collection + POST to `kensho/goals`; the three AI flows all routed through the privacy modal (populate "what will be sent", pick provider, confirm → fetch JSON endpoint → render editable result → user confirms into the relevant fields). Uses `fetch` with the reusable CSRF token from `data-csrf`, `credentials: 'same-origin'`, and posts `application/x-www-form-urlencoded` bodies including `csrf_token`.

```javascript
/* Kensho — buildless dashboard behavior. No inline script (CSP default-src 'self'). */
(function () {
    'use strict';

    function boot() {
        var root = document.querySelector('[data-kensho-root]');
        if (!root) { return; }

        var csrf = root.getAttribute('data-csrf') || '';
        var urls = {
            propose: root.getAttribute('data-url-propose'),
            clarify: root.getAttribute('data-url-clarify'),
            reflect: root.getAttribute('data-url-reflect')
        };

        setupTabs(root);
        setupActions(root, csrf, urls);
    }

    function setupTabs(root) {
        var tabs = root.querySelectorAll('[data-kensho-tab]');
        tabs.forEach(function (tab) {
            tab.addEventListener('click', function () {
                var name = tab.getAttribute('data-kensho-tab');
                root.querySelectorAll('[data-kensho-panel]').forEach(function (p) {
                    p.hidden = p.getAttribute('data-kensho-panel') !== name;
                });
            });
        });
    }

    function post(url, data, csrf) {
        var body = new URLSearchParams();
        body.set('csrf_token', csrf);
        Object.keys(data).forEach(function (k) { body.set(k, data[k]); });
        return fetch(url, {
            method: 'POST',
            credentials: 'same-origin',
            headers: { 'Content-Type': 'application/x-www-form-urlencoded' },
            body: body.toString()
        }).then(function (r) { return r.json(); });
    }

    function openPrivacy(root, summaryText, onConfirm) {
        var modal = root.querySelector('[data-kensho-privacy]');
        if (!modal) { onConfirm(''); return; }
        modal.querySelector('[data-kensho-privacy-detail]').textContent = summaryText;
        modal.hidden = false;

        var confirm = modal.querySelector('[data-kensho-privacy-confirm]');
        var cancel = modal.querySelector('[data-kensho-privacy-cancel]');
        function cleanup() { modal.hidden = true; confirm.removeEventListener('click', ok); cancel.removeEventListener('click', no); }
        function ok() {
            var sel = modal.querySelector('[data-kensho-provider]');
            cleanup();
            onConfirm(sel ? sel.value : '');
        }
        function no() { cleanup(); }
        confirm.addEventListener('click', ok);
        cancel.addEventListener('click', no);
    }

    function setupActions(root, csrf, urls) {
        root.addEventListener('click', function (e) {
            var btn = e.target.closest('[data-kensho-action]');
            if (!btn) { return; }
            var action = btn.getAttribute('data-kensho-action');

            if (action === 'propose') {
                openPrivacy(root, 'Aggregated dimensions + up to 5 sample titles each.', function (provider) {
                    post(urls.propose, { provider_profile: provider }, csrf).then(function (res) {
                        renderProposedAreas(root, res);
                    });
                });
            } else if (action === 'clarify') {
                var label = btn.getAttribute('data-area-label') || '';
                var fieldset = btn.closest('[data-area-id]');
                var note = fieldset ? fieldset.querySelector('[data-area-identity]').value : '';
                openPrivacy(root, 'Area: ' + label + '\nYour note: ' + note, function (provider) {
                    post(urls.clarify, { area_label: label, note: note, provider_profile: provider }, csrf).then(function (res) {
                        applyClarified(fieldset, res);
                    });
                });
            } else if (action === 'reflect') {
                openPrivacy(root, 'Your discrepancies + your desired-self statements.', function (provider) {
                    post(urls.reflect, { provider_profile: provider }, csrf).then(function (res) {
                        applyReflection(root, res);
                    });
                });
            } else if (action === 'save-areas') {
                saveAreas(root, csrf);
            } else if (action === 'save-goals') {
                saveGoals(root, csrf);
            }
        });
    }

    // Render AI-proposed areas as editable rows; nothing persists until "Save areas".
    function renderProposedAreas(root, res) {
        var editor = root.querySelector('[data-kensho-areas-editor]');
        if (!editor || !res || !res.areas) { return; }
        editor.innerHTML = '';
        res.areas.forEach(function (a) {
            var row = document.createElement('div');
            row.className = 'kensho-area';
            row.setAttribute('data-proposed', '1');
            var input = document.createElement('input');
            input.type = 'text';
            input.value = a.label;
            input.setAttribute('data-proposed-label', '1');
            row.appendChild(input);
            row.setAttribute('data-refs', JSON.stringify(a.dimension_refs || []));
            if (a.importance_hint) { row.setAttribute('data-importance', String(a.importance_hint)); }
            var note = document.createElement('p');
            note.className = 'kensho-hint';
            note.textContent = a.rationale || '';
            row.appendChild(note);
            editor.appendChild(row);
        });
    }

    function saveAreas(root, csrf) {
        var editor = root.querySelector('[data-kensho-areas-editor]');
        var areas = [];
        if (editor) {
            editor.querySelectorAll('[data-proposed]').forEach(function (row) {
                var label = (row.querySelector('[data-proposed-label]') || {}).value || '';
                if (!label.trim()) { return; }
                var refs = JSON.parse(row.getAttribute('data-refs') || '[]');
                var sources = { project_ids: [], category_ids: [], tag_ids: [] };
                refs.forEach(function (r) {
                    if (r.type === 'project') { sources.project_ids.push(r.id); }
                    else if (r.type === 'category') { sources.category_ids.push(r.id); }
                    else if (r.type === 'tag') { sources.tag_ids.push(r.id); }
                });
                areas.push({ label: label.trim(), importance: parseInt(row.getAttribute('data-importance') || '5', 10), sources: sources });
            });
        }
        submitForm(root, 'KenshoController', 'areas', { areas: JSON.stringify(areas) }, csrf);
    }

    function applyClarified(fieldset, res) {
        if (!fieldset || !res) { return; }
        if (res.identity) { fieldset.querySelector('[data-area-identity]').value = res.identity; }
        if (res.goals && res.goals.length) { fieldset.querySelector('[data-area-goals]').value = res.goals.join('\n'); }
    }

    function saveGoals(root, csrf) {
        var updates = [];
        root.querySelectorAll('[data-area-id]').forEach(function (fs) {
            updates.push({
                id: fs.getAttribute('data-area-id'),
                importance: parseInt(fs.querySelector('[data-area-importance]').value, 10),
                identity: fs.querySelector('[data-area-identity]').value,
                goals: fs.querySelector('[data-area-goals]').value.split('\n').map(function (s) { return s.trim(); }).filter(Boolean)
            });
        });
        submitForm(root, 'KenshoController', 'goals', { goals: JSON.stringify(updates) }, csrf);
    }

    function applyReflection(root, res) {
        if (!res) { return; }
        var parts = [];
        (res.observations || []).forEach(function (o) { parts.push('• ' + o); });
        (res.questions || []).forEach(function (q) { parts.push('? ' + q); });
        if (res.gentle_note) { parts.push('\n' + res.gentle_note); }
        var ta = root.querySelector('[data-kensho-narrative]');
        if (ta) { ta.value = parts.join('\n'); }
    }

    // Build + submit a real form (full-page redirect keeps the CSRF/flash flow simple).
    function submitForm(root, controller, action, fields, csrf) {
        var form = document.createElement('form');
        form.method = 'post';
        // The dashboard root exposes each action URL via its rendered forms; fall back to query-string form.
        form.action = '?controller=' + controller + '&action=' + action + '&plugin=Kensho';
        var all = Object.assign({ csrf_token: csrf }, fields);
        Object.keys(all).forEach(function (k) {
            var i = document.createElement('input');
            i.type = 'hidden'; i.name = k; i.value = all[k];
            form.appendChild(i);
        });
        document.body.appendChild(form);
        form.submit();
    }

    if (document.readyState === 'loading') {
        document.addEventListener('DOMContentLoaded', boot);
    } else {
        boot();
    }
})();
```

> The reflection save uses the real `<form data-kensho-reflection-form>` already in `reflection.php`; before submit, JS sets its `data-kensho-discrepancies` hidden input from the current alignment rows if you want the snapshot to carry them (optional — the controller tolerates empty). Keep all endpoints reachable via the query-string form (`?controller=...&action=...&plugin=Kensho`) so they work over curl too.
