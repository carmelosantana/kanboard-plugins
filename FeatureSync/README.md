# FeatureSync

A Kanboard plugin that bulk-copies project features — automated actions, tags, and columns — from a source project to many target projects in one operation.

## Features

- Pick a source project as template
- Choose which features to copy (actions, tags, columns)
- Select multiple target projects
- Preview changes before applying
- Apply in add-missing or replace mode

## Requirements

- Kanboard >= 1.2.47
- PHP >= 8.4

## Installation

Copy or symlink the `FeatureSync/` directory into your Kanboard `plugins/` directory.

## Usage

Navigate to **Settings → Feature Sync** (admin only).
